# Perfume Website UI

A modern perfume website UI built using HTML and CSS. This project was created to practice front-end web design, page layout, and clean user interface development.

## Features

- Modern landing page design
- Hero section
- Product showcase section
- Elegant and clean UI
- HTML5 & CSS3 based layout

## Technologies Used

- HTML5
- CSS3

## Project Purpose

This project was built as a front-end practice project to improve HTML and CSS skills and create a visually appealing website interface.
